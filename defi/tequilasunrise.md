# Tequila Sunrise

- A wearable warm wallet for spending, stable cons, etc.
    - Similar to the [decoy wallet](singaporesling.md)
    - But focuses on stable coins instead of ETH, ADA, SOL, etc
- Uses [Aperol Spritz](aperolspritz.md) as a secure stash
    - [Aperol Spritz](aperolspritz.md) only communicates with Tequila Sunrise
- Ingress/Egress from/to CEXs
- Trampoline for Singapore Sling DeFi when needed
- Does not currently use passphrase.

- [Seed](tequilasunrise.uuen)
- [Passphrase](tequilasunrise25.uuen)

- Seed and Passphrase are encrypted with some **medium** meandering passphrase
    - Obtain that medium meandering passphrase
    - Find the [Recipe](caving) and **short** meandering passphrase
    - Decrypt the [seed](tequilasunrise.uuen) to get the seed phrase
    - Restore to a new wallet
